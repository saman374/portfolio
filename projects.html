Below are hardware + software projects I built ( each card explains both the electronics and the code.)

Neera Extraction System
A semi-automated neera collection and processing prototype that improves hygiene and collection efficiency.
  • Hardware: Raspberry Pi Pico / Arduino Nano, peristaltic pump, level sensors, temperature sensor, stainless-steel collection chamber, relay module, 12V power supply, and simple mechanical holder for coconut inflorescence.
  • Software: Embedded firmware to control pump cycles and read sensors; a Python/Flask dashboard (runs on local Pi) to view real-time levels, set collection schedules, and log yields.
Key features: timed extraction cycles, overflow protection, simple web UI for monitoring, and CSV export for yield tracking.

A full-stack automatic watering system that waters plants based on soil moisture and schedule, with remote monitoring.
